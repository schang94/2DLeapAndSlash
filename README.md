# 2D Leap & Slash
전방에서 나오는 적과 공격을 피해 생존
## 개발
- 엔진 : Unity
- 언어 : C#
- 주요 에셋/패키지 : Input System

## 조작방법
- 이동 : 방향키
- 점프 : 스페이스바 (2단점프)
- 공격 : A키

## 게임 특징
- 4가지의 캐릭터가 있고 캐릭터 마다 공격속도, 공격력이 다르게 설정이 되어있다.
- 적은 두가지로 구성되어 있는다. 첫번째는 앞으로 직진하고 두번째는 직진하다가 중간에 점프를 한다.
- 위쪽에서 날아오는 장애물과 적을 피해하고 적을 공격하여 제거하여 생존하는 게임이다.
